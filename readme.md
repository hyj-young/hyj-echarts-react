# hyj-echarts-react

## install

use `npm install hyj-echarts-react` to install

## dependencies && versions

1. react: 15+
2. echarts: 4.1.0
3. hyj-func: 1.6.0

## 使用示例

`git clone https://github.com/hyj-young/hyj-echarts-react.git`

`npm install`

`npm run dev`

test dir: `src/test.js`

## api

option: echarts option

events: echarts events
